# Грамматика: поиск транзитивных зависимостей в графе модулей
# Находит все пары (A, B) такие, что модуль A прямо или транзитивно зависит от B
# Используется с графом software_deps.csv
# Стартовый нетерминал: Dep
#
# dependsOn — прямая зависимость
# imports   — импорт (тоже считается зависимостью)
Dep -> dependsOn
Dep -> imports
Dep -> dependsOn Dep
Dep -> imports Dep
