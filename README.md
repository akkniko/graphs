#   Исследование:  Algebraic-CFPQ-Engine
## 1. Введение и предмет исследования
Работа про исследование задачи **Context-Free Path Querying (CFPQ)** - поиск пар вершин $(u, v)$ в направленном помеченном графе, пути между которыми формируют слова, принадлежащие заданному контекстно-свободному языку

-----
\
## 2. Ключевая идея исследования
Основная идея заключается в представлении топологии графа и правил грамматики в виде матриц смежности и их последующей итеративной обработке до достижения фиксированной точки
----
## 3. Рассмотренные алгоритмы
В работе сделаны 2 фундаментальных алгебраических алгоритма CFPQ:

### 3.1. Матричный алгоритм (на основе WCNF)
* **Источник:** Диссертация Азимова, Глава 3, раздел 3.1 (Листинг 4)
* **Суть:** Входная КС-грамматика приводится к ослабленной нормальной форме Хомского (WCNF). Для каждого нетерминала $A$ инициализируется булева матрица $T_A$. На каждом шаге для всех бинарных правил $A \rightarrow B \ C$ вычисляется композиция путей через матричное умножение: $T_A = T_A \oplus (T_B \times T_C)$.

### 3.2. Тензорный алгоритм (на основе произведения Кронекера)
* **Источник:** Диссертация Азимова, Глава 4, раздел 4.1
* **Суть:** Избавляет от необходимости трансформировать грамматику. Правила представляются в виде рекурсивного автомата (RSM). На каждой итерации вычисляется произведение Кронекера матриц переходов автомата и матриц смежности графа, после чего рассчитывается транзитивное замыкание объединенной структуры для одновременного трекинга состояний автомата и шагов по графу.

---

## 4. Архитектура программного комплекса
структура проекта:
* `grammar_utils.py` - парсинг текстовых грамматик и трансформации их в форму WCNF (бинаризация)
* `cfpq_matrix.py` - инкапсуляция матричного алгоритма
* `cfpq_kronecker.py` реализация построения RSM, вычисления тензорного произведения Кронекера и транзитивного замыкания
* `main.py` - точка входа, также есть тестовые сценарии

* **Конфигурация:** Граф представляет собой структуру из двух пересекающихся в узле `0` циклов (длиной 3 по метке `a` и длиной 2 по метке `b`). Грамматика: $L = \{a^n b^n \mid n \ge 1\}$.
